How to Use

  1. Parse MIB Files: First, run the MIB parser to convert your .mib files into JSON format: 
    python parse_mib_files.py
  2. Start the Application: Launch the Flask web application: 
    python -m flask_app.app
  3. View MIBs: Open your browser and navigate to http://localhost:5000
  4. Explore: Browse MIB files, search for nodes, and interact with tree visualizations.

Keyboard Shortcuts

  • Ctrl+K or Cmd+K - Focus search
  • Escape - Clear search
  • Enter - Submit search
  • Click node - Expand/collapse tree
  • Hover node - Show tooltip
  • Back button - Navigate back
